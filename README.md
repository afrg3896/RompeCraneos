# RompeCraneos
Videojuego Educativo que permite Desarrollar y mejorar habilidades de pensamiento cómo la capacidad de concentración y planificación, el razonamiento y la comprensión, la vista y la imaginación espacial. Aprender y crear estrategias que inciten el desarrollo del pensamiento lógico, además de crear hábitos para el uso adecuado del tiempo libre.
